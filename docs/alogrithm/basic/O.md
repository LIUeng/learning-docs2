# 大O表示法

- 所有代码执行的时间T(n)与每行代码执行次数n成正比
  ```
    T(n) = O(f(n))
  ```
## 时间复杂度
  `大O表示的并不是程序执行的时间，而是代码执行时间随着数据增长的变化趋势，渐进时间复杂度`
  
  1. 只关注循环最多的那一段代码
  2. 加法法则：总复杂度等于量级最大的那段代码的复杂度
  3. 乘法法则：嵌套代码的复杂度等于嵌套内外复杂度的乘积
